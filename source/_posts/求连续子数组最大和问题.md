### 问题描述

在一个数组中（数组中的元素可以是负数），找出一个连续的子数组，要求子数组中的元素相加，和最大；然后输出这个最大值和最大子数组。

举例：

数组A = {1，3，-5，7，6，-1，3}，数组A的和最大的连续子数组是{7，6，-1，3}，最大和是15.

### 思路

- 先将这个数组中所有的连续子数组都找出来
- 再将每个子数组中的元素求和
- 比较出最大和的子数组

### 代码

```java
/**
 * Created by Damon-zln on 2019/2/19.
 */
public class Test {

    public static void main(String[] args) throws Exception{
        int[] arr = {1, 2, 3};
        System.out.println(findAllSubArray(arr));
    }

    private static StringBuilder findAllSubArray(int[] arr) {
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < arr.length; i++) {
            List<Integer> list = new LinkedList<>();
            for (int j = i; j < arr.length; j++) {
                list.add(arr[j]);
                sb.append(list).append("|");
            }
        }
        return sb;
    }
}
--------------------------------------------------------------------------------
执行结果：

到这里，我的想法是，既然我们已经将子数组找出来了，那么我们可以在找子数组的时候，将子数组的元素和也算出来，有了子数组的和之后，就可以再添加一个判断，筛选出和最大的子数组，完美！！！
具体代码如下：
================================================================================
/**
 * Created by Damon-zln on 2019/2/19.
 */
public class Test {

    public static void main(String[] args) {
        int[] arr = {1, 3, -5, 7, 6, -1, 3};
        System.out.println(findMaxSubArray(arr));
    }

    /**
     * 连续子数组的最大和
     * @param arr 输入的数组
     * @return map ：key是最大和，value是最大和对应的子数组
     */
    private static Map<Integer, StringBuilder> findMaxSubArray(int[] arr) {
        int sum = 0;
        // 先将数组第一位赋值给max
        int max = arr[0];
        // 初始化一个map，key用来存放最大连续子数组的和，value用来存放最大连续子数组
        Map<Integer, StringBuilder> map = new HashMap<>();
        // 初始化StringBuilder，用来存放和最大的子数组
        StringBuilder sb = new StringBuilder();
        // 第一个循环用来确定子数组开始的位置
        for (int i = 0; i < arr.length; i++) {
            sum = 0;
            // 初始化一个集合用来存放子数组的元素
            List<Integer> list = new LinkedList<>();
            for (int j = i; j < arr.length; j++) {
                // 子数组内元素求和
                sum += arr[j];
                // 存放子数组的元素
                list.add(arr[j]);
                // 判断比较当前子数组元素和是不是最大
                if (sum > max) {
                    max = sum;
                    // 删除之前存放的子数组
                    sb.delete(0, sb.length());
                    // 添加当前的子数组
                    sb.append(list);
                }
            }
        }
        map.put(max, sb);
        return map;
    }
}

================================================================================
/**
 * Created by Damon-zln on 2019/2/19.
 */
public class Test {

    public static void main(String[] args) throws Exception{
        int[] arr = {1, 3, -5, 7, 6, -1, 3};
        System.out.println(findMaxSubArray(arr));
    }

    private static Map<Integer, StringBuilder> findMaxSubArray(int[] arr) {
        int sum = 0;
        int max = arr[0];
        Map<Integer, StringBuilder> map = new HashMap<>();
        StringBuilder sb = new StringBuilder();
        List<Integer> list = new LinkedList<>();
        for (int i = 0; i < arr.length; i++) {
            sum += arr[i];
            list.add(arr[i]);
            if (sum > max) {
                max = sum;
                sb.delete(0, sb.length());
                sb.append(list);
            // 如果累加和出现小于0的情况, 则和最大的子序列肯定不可能包含前面的元素
            // 这时将累加和置0，从下个元素重新开始累加
            } else if (sum < 0) {
                sum = 0;
                list = new LinkedList<>();
            }
        }
        map.put(max, sb);
        return map;
    }
}
--------------------------------------------------------------------------------
执行结果：
{15=[7, 6, -1, 3]}
```
